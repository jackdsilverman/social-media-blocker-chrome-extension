# Social Media News Feed Blocker

This Chrome extension blocks the news feed, Facebook Reels, Facebook Stories, YouTube Shorts, and YouTube ads for a cleaner and distraction-free browsing experience on Facebook, YouTube, and LinkedIn.

## Features

- Automatically hides:
  - News feed, Reels, and Stories on Facebook.
  - Home feed, Shorts (both section and tab), and ads on YouTube.
  - News feed on LinkedIn.
- Lightweight and easy to install.

## Installation

1. Clone the repository:
2. Open chrome://extensions/ in your chrome browser
3. Enable developer mode toggle in top right of the page
4. Click 'Load unpacked' button
5. select the src folder.
6. verify extension works!
